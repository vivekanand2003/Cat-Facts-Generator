# 🐱 Cat Facts Generator (API Project)

A simple web project that fetches random cat facts from a public API and displays them on the page when the user clicks a button.
This project demonstrates basic API integration using **Axios**, **HTML**, **CSS**, and **JavaScript**.

---

## 🚀 Features

* Fetches random cat facts from an online API
* Displays fact instantly on button click
* Handles API errors gracefully
* Clean and simple UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Axios (for API requests)

---

## 📂 Project Structure

```
Cat-Facts-API/
│── Api.html
│── api.js
```

* **Api.html** → Main webpage structure and UI
* **api.js** → API request and logic

---

## 🌐 API Used

Cat Facts API: [https://catfact.ninja/fact](https://catfact.ninja/fact)

The app sends a GET request to this endpoint and retrieves a random cat fact. 

---

## ▶️ How to Run the Project

1. Download or clone this repository
2. Open `Api.html` in your browser
3. Click **“Click Here To Get Cat Facts”**
4. A random cat fact will appear

---

## 💡 How It Works

* A button click triggers an async function
* Axios sends a GET request to the Cat Facts API
* The response fact is displayed inside the page

The button and result container are defined in the HTML file.
